#  Score for Good App Dashboard

##  Project Overview
The Score for Good App Dashboard is a full-stack web application developed to manage and track user scores efficiently. It provides an interactive interface where users can add scores, view their history, and analyze performance through visual charts.

---

##  Features
- Add and manage scores with date tracking  
- View recent score history  
- Calculate total, latest, and average scores  
- Visualize data using bar and line charts  
- Responsive UI built with Material UI  
- Efficient state management using Redux Toolkit  
- API handling and caching using RTK Query  

---

##  Tech Stack

**Frontend:**
- React.js  
- Material UI  

**State Management:**
- Redux Toolkit  
- RTK Query  

**Data Visualization:**
- Recharts  

**Backend:**
- Node.js  
- Express.js  

**Database:**
- MongoDB  



## ⚙️ How It Works
- Users enter scores along with dates  
- Data is stored in the database (MongoDB)  
- Dashboard calculates key metrics (total, average, latest)  
- Charts display score trends and distribution
